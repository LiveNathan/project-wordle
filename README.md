# Word Game (Wordle Clone)

This is my completed homework submission for **Project I: Word Game** from
[Josh Comeau's Joy of React course](https://courses.joshwcomeau.com/joy-of-react).

**This is not the original project repo.** The starter code, exercises, and
assets belong to Josh Comeau; the original project template lives at
[joy-of-react/project-wordle](https://github.com/joy-of-react/project-wordle).
This repo contains my own implementation of the exercises and stretch goals
built on top of that starter.

## What's implemented

- Exercise 1: Guess input form, controlled, uppercased, validated to 5 letters
- Exercise 2: Tracking and rendering submitted guesses
- Exercise 3: Fixed 6x5 guess grid
- Exercise 4: Letter status (correct / misplaced / incorrect) via `checkGuess`
- Exercise 5: Win/loss banners, input disabled once the game ends

Stretch goals (visual keyboard, restart button) were not attempted.

## Running locally

```bash
npm install
npm run dev
```

See the [course's local dev instructions](https://courses.joshwcomeau.com/joy-of-react/project-wordle/03-dev-server)
if you run into setup issues.
